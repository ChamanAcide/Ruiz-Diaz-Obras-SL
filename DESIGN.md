# DESIGN SYSTEM

## Section 6: Design System Notes for Stitch Generation
**Aesthetics**: Haute Façon - Luxury Minimal.
**Colors**:
- Primary background: #FAFAFA (very light gray/white)
- Text color: #111827 (dark slate)
- Accents: #4B154C (deep plum/burgundy for luxury feel) and #D4AF37 (subtle gold accents if needed).
**Typography**: Inter (modern, clean, sans-serif).
**Components**:
- Glassmorphism effects on cards (subtle blur, semi-transparent white backgrounds with delicate borders).
- Bento Grid layouts for services and features to organize content cleanly.
- Buttons should have smooth hover animations and distinct accent colors.
**Layout**: Ensure high contrast, ample whitespace, and strict structural components (Headers, Heroes, Footers must be unified). All interactive elements must feel premium.
